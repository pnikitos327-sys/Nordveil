# NORDVEIL — Clothing Brand Website

![Preview](images/screenshot.jpg)

Современный лендинг для бренда одежды с 3D-графикой и анимациями.

##  Стек технологий
- HTML5
- CSS3 (Flexbox, Grid, анимации)
- JavaScript (ES6+)
- Three.js (3D частицы)
- GSAP (плавные анимации)

##  Функционал
- 3D-сцена с вращающейся сферой и частицами
- Карусель товаров
- Фильтрация по категориям
- Избранное (сохраняется в localStorage)
- Таймер ограниченной серии
- Адаптивный дизайн
- Плавный скролл

##  Запуск проекта
1. Склонировать репозиторий
2. Открыть `index.html` в браузере
